# Changelog

All notable changes to the Adapto VS Code extension.

## [0.1.0] — 2026-05-28

### Added

- TextMate grammar for `.adapto` files with embedded Rust, CSS, and HTML support
- LSP-based diagnostics, completions, hover, go-to-definition, document symbols, semantic tokens, and formatting
- 12 code snippets (apage, aroute, ascript, aresource, alayout, aif, aifelse, aeach, amatch, acan, aaction, aform)
- CLI integration: dev server, build, check, generate, routes, doctor commands
- Project sidebar with routes, components, and resources tree view
- Live preview WebView panel
- Block indicator status bar icons
- Control flow pair highlighting ({#if}/{/if}, {#each}/{/each}, etc.)
- Security lens inline annotations for #[permission] and #[audit]
- Visual block separators between top-level blocks
- Adapto Night (dark) and Adapto Day (light) color themes
- Extension settings for all configurable features
